# Security Policy

The FlowWright team takes security seriously across all
[Flow-Wright](https://github.com/Flow-Wright) repositories.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues,
discussions, or pull requests.**

Instead, email **ezequiel.migueles@flowwright.dev** with:

- A description of the vulnerability and its impact.
- Steps to reproduce, or a proof of concept.
- The affected repository, version, or commit.

You'll get an acknowledgement as quickly as we can manage. We'll work with you on a fix
and coordinate a disclosure timeline before any public write-up.

## Supported versions

FlowWright is pre-launch. Until the first tagged release, only the `main` branch of each
repository is supported.

Thanks for helping keep FlowWright and its users safe. 🦫
