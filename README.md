# .github

Org-wide defaults for [**Flow-Wright**](https://github.com/Flow-Wright). GitHub reads
this special repository to configure the organization.

- [`profile/README.md`](profile/README.md) — the public org landing page shown on
  [github.com/Flow-Wright](https://github.com/Flow-Wright).
- [`CONTRIBUTING.md`](CONTRIBUTING.md), [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md),
  [`SECURITY.md`](SECURITY.md) — default community health files, inherited by every repo
  that doesn't ship its own.
- [`.github/`](.github) — default issue forms, PR template, and funding config.
- [`templates/REPO_README.md`](templates/REPO_README.md) — starter README for new
  Flow-Wright repos.
