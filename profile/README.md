**FlowWright is a self-hosted control plane for your pipelines** — see every run,
follow every stage live, and keep the whole record. Author pipelines as typed
TypeScript instead of YAML, run them the same way on your laptop, on any CI provider,
or on an optional team server. Open source, MIT licensed.

### Why FlowWright

- **Typed pipelines, not YAML** — author runs as TypeScript code, with real types and refactoring.
- **Local-first & portable** — the same pipeline runs on your machine, on any CI, or on a team server.
- **Live run visibility** — watch every run and follow each stage as it happens.
- **Open source, MIT** — self-hosted, no lock-in, keep the whole record.
